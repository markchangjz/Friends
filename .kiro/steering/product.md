# 產品概述

## Friends App - 好友列表頁面 iOS 實作

一個好友列表管理 iOS 應用程式，展示現代 iOS 開發實務，採用 MVVM 架構與 Combine 框架。

### 核心功能
- **動態頁面狀態**：處理「無好友」、「僅好友列表」、「好友列表含邀請」三種情境
- **並行資料載入**：使用 async/await 並行呼叫 API，提升啟動效能
- **智慧資料整合**：自動合併重複的好友資料，保留基於 updateDate 的最新版本
- **即時搜尋**：支援關鍵字即時過濾好友
- **下拉更新**：隨時重新取得最新資料

### 進階互動與動畫
- **堆疊卡片式邀請列表**：支援展開/收合的流暢動畫效果
- **搜尋轉場動畫**：模擬原生 SearchBar 的流暢轉場
- **骨架屏載入**：資料載入期間的優雅漸層動畫
- **徽章通知**：自動計算待處理好友邀請數量並顯示於標籤頁
- **空狀態視覺**：無資料時的精美插畫佔位

### 技術亮點
- **100% 原生 UI**：未使用任何第三方 UI 函式庫，所有自訂 TabBar 與動畫皆手動實作
- **深色模式支援**：所有 UI 元件完美適配淺色/深色模式
- **現代並發處理**：使用 async/await 與 Task Groups 處理複雜的非同步邏輯
- **設計系統**：透過 DesignConstants 統一管理字體、間距與顏色
